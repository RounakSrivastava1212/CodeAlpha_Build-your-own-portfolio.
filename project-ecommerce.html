E‑commerce UI

A clean user interface for an online store, featuring product cards, filters, and a detailed product view.

HTML5 CSS Grid UI Design

Overview

This project explores the layout and interaction patterns of an online shop. It includes a product grid, basic filtering UI, and a product detail layout with images, price, and description.

What I focused on

  • Using CSS Grid to create a flexible product listing layout.
  • Designing clean product cards with consistent spacing and hierarchy.
  • Adding hover states and focus styles for interactive elements.
  • Preparing the layout so it can later be hooked to real data.

Tech stack

  • Semantic HTML for structure.
  • CSS Grid and Flexbox for layout.
  • Basic JavaScript for UI interactions (filter toggles, etc.).
