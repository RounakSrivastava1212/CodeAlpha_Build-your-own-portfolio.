Responsive Landing Page

A modern, mobile‑first landing page focused on clean layout, clear hierarchy, and smooth scrolling interactions.

HTML5 CSS3 Responsive Design Flexbox

Overview

This project is a single‑page marketing site designed to look great on phones, tablets, and desktops. It includes a hero section, features, call‑to‑action, and a simple pricing layout, all built with semantic HTML and modern CSS.

What I focused on

  • Building a pixel‑perfect layout using Flexbox and CSS Grid.
  • Making sure the design is fully responsive from 320px to large screens.
  • Using reusable utility classes for spacing and typography.
  • Adding smooth scrolling and subtle hover effects.

Tech stack

  • HTML5 for structure and accessibility.
  • CSS3 with Flexbox and Grid for layout.
  • Basic JavaScript for interaction (scroll behavior, minor effects).
